# Class Restrictions
Highlander runs on a maximum of one person per class. Scout is set to unlimited to allow for players to swap around classes without switching to spec.
1 - Scout (unlimited for players to swap if needed)
1 - Soldier
1 - Pyros
1 - Demoman
1 - Heavy
1 - Engineer
1 - Medic
1 - Sniper
1 - Spy
